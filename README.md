# yt-dl
youtube video/audio downloader


> only youtube links are supported

## How to Run :
```
# clone repo
git clone https://github.com/SoniaCore/yt-dl.git
cd yt-dl

# install requirements
pip install -r requirements.txt
```
```
# run ytbot
ytbot.py 'https://youtu.be/EngW7tLk6R8'
```
